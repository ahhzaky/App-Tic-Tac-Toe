# App-Tic-Tac-Toe

### Install this project

```
npm install
```

### Start Project

```
npm run start
```

### Make to application distribution

```
npm run make
```

You can check the application in the folder out/

### Build with

- [ElectronJs](https://www.electronjs.org/)
